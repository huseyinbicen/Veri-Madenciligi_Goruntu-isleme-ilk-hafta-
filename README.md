Ödev: Bir fotoğrafın istatisliksel değerleri hesaplayan bir uygulam yapınız.
Ben pixel ler üzerindde bulunsan R,G,B değerleri bir listeye yerleştirip istatisliksel değerler olan;
* Mod
* Medyan
* Ortalama
* Varyans
* Standart Sapma
* Çarpıklık
* Basıklık
* Değişim Kat sayısı
* Max
* Min
